JS XtraLife SDK Sample

You're currently browsing a sample showing some of the XtraLife Javascript SDK's social features. Feel free to use/modify your local copy of this code for your own purposes.

Important! Before anything, you need to set your game's API Key and Secret at the beginning of the "sample.js" file (in the "js" folder of the present sample) in the corresponding variables.

To avoid getting errors, you should get the latest version of the Javascript SDK from our Github

The "JS client" can be used both from node.js or from a web browser. It's written in Javascript and compiled with Browserify and optionnally minified to a mere 36kB, less than 8kB after gzip compression.

Note: This client is not feature complete yet. It implements the latest API. Feel free to submit bug reports and/or pull requests.

To understand our features, please see the documentation of our backend.
